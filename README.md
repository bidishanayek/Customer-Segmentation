
# Customer Segmentation Analysis

## Project Objective

The objective of this project is to segment customers based on **demographic, behavioral, and transaction features** to help businesses design personalized marketing campaigns, improve customer retention, and identify high-value customer groups.

---

## Features Used

### Demographic Features

* Age
* Gender
* City / Region
* Occupation
* Income Level

### Behavioral Features

* Website visits per month
* App engagement score
* Email response rate
* Product category interest score

### Transaction Features

* Purchase frequency
* Average order value
* Total annual spending
* Recency (days since last purchase)
* Discount usage percentage
* Return rate

---

## Customer Segments Identified

| Segment                      | Key Characteristics                            | Business Strategy               |
| ---------------------------- | ---------------------------------------------- | ------------------------------- |
| Premium Loyal Customers      | High income, high spending, frequent purchases | Loyalty rewards, premium offers |
| Discount-Driven Buyers       | High discount usage, moderate spending         | Coupon campaigns                |
| Occasional Customers         | Low purchase frequency                         | Engagement campaigns            |
| New / Low Activity Customers | Low engagement and spending                    | Onboarding promotions           |

---

## Business Impact

* Identified top revenue-contributing customers
* Reduced marketing cost by targeting the right segments
* Enabled personalized campaigns increasing engagement
* Built reusable segmentation pipeline for monthly CRM updates
